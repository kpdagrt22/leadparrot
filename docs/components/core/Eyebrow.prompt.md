Mono uppercase section kicker. Always pair above a serif headline to establish the Crest editorial rhythm.

```jsx
<Eyebrow>How it works</Eyebrow>
<h2>From public conversation to helpful reply</h2>

<Eyebrow tone="accent" tick={false}>Live</Eyebrow>
```

Set `tick={false}` to drop the leading rule. `tone="accent"` for active/live contexts.
