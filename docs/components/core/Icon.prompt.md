Lucide icon wrapper — the Crest icon standard (1.75px stroke, line style). Page must load the Lucide UMD script (`window.lucide`).

```jsx
<Icon name="search" size={16} />
<Icon name="arrow-up-right" size={14} color="var(--accent)" />
```

Use sparingly and at a consistent stroke. Prefer line icons over filled. Never substitute emoji.
