Squared identity mark with mono initials. Use for orgs, post authors (u/handle), workspace switcher.

```jsx
<Avatar label="Acme Proposals" tone="accent" />
<Avatar label="u/design_freelance" size={28} tone="soft" />
<Avatar src="/logo.png" label="Acme" shape="pill" />
```

`shape="square"` (default, on-brand) or `"pill"`. Tones: `ink · accent · soft`. Derives up to 2 initials from `label`.
