Flat callout for safety / disclosure / status — left ink rule + tinted ground (no rounded "alert" look). The Crest voice for LeadParrot's platform-safety and disclosure notes.

```jsx
<Notice tone="caution" title="Platform safety">Check each community's rules before posting.</Notice>
<Notice tone="accent" title="Suggested disclosure">I'm building a tool in this space…</Notice>
```

Tones: `info · accent · caution · danger · neutral`. Optional `title` (mono) and `icon`.
