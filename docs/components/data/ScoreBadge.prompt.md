LeadParrot intent score as an editorial unit — serif number + mono tier label. Tiers: high ≥70, medium 40–69, low <40.

```jsx
<ScoreBadge score={81} size="lg" />
<ScoreBadge score={63} showTier={false} />
```

Color tracks the tier automatically (green/ochre/clay). Pair with `ScoreBars` for the weighted breakdown.
