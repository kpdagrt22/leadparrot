Thin squared usage / progress bar with mono numerals. Auto-switches the fill to clay past 90% (limit warning).

```jsx
<Meter label="Posts scanned" value={342} max={500} />
<Meter label="Reply drafts" value={88} max={100} />
```

Tones: `accent · soft · medium · low · ink`. Set `showValue={false}` for a bare gauge.
