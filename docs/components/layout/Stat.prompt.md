Editorial metric — mono uppercase label over a large serif number. The dashboard's data voice.

```jsx
<Stat label="High-intent" value="41" hint="score ≥ 70" />
<Stat label="Total leads" value="248" trend={{ dir: "up", value: "12" }} />
```

`size`: `sm|md|lg`. `align`: `left|center`. `trend.dir`: `up` (green) / `down` (clay) / `flat`.
